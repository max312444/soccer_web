# サッカー情報掲示板 Web アプリケーション

本プロジェクトは、サッカーをテーマとした  
情報提供型の掲示板 Web アプリケーションです。  
外部 API を活用して  
サッカー関連情報を取得・表示することに重点を置いて開発しました。

ユーザーは簡単な会員登録とログインを通じてサービスを利用でき、  
ログイン後のみすべての機能にアクセスできる構成としています。

---

## プロジェクト概要

- プロジェクト種別：サッカー情報掲示板型 Web アプリケーション
- 開発目的  
  外部 API と連携し、取得したデータを  
  Web 画面上に構造的に表示する一連の流れを学習すること
- 主な特徴
  - ログインベースのサービス利用制御
  - ユーザーの好みのチーム設定
  - サッカーニュース、リーグ順位、試合日程 API の活用

## ユーザー利用フロー

1. 会員登録  
   - ID・パスワードによる基本的な会員登録  
   - 登録時に好みのチームを選択可能
2. ログイン  
   - ログイン成功時にユーザー情報を読み込み  
   - ログインボックスに好みのチームのロゴを表示
3. ログイン後のみ全メニュー利用可能  
   - 未ログイン状態ではサービスメニューへのアクセスを制限

---

## メイン画面構成

メイン画面は、情報を効率よく確認できるよう  
3 カラムレイアウトで構成しています。

- 左側エリア  
  - ログインボックス  
  - ログイン時にユーザーの好みのチームロゴを表示

- 中央エリア  
  - サッカー関連の最新ニュースを表示  
  - 外部ニュース API からデータを取得

- 右側エリア  
  - リーグ別の現在 1 位チーム情報を表示  
  - チーム名および勝点を表示

※ 画面イメージは後日追加予定

## 主な機能

### ログイン / 会員登録

- 基本的な会員登録およびログイン機能
- 会員登録時の好みのチーム選択
- ログイン状態に応じた機能アクセス制御

---

### リーグ順位表示

- 主要サッカーリーグの順位確認機能
- 合計 6 つの代表的なリーグを提供
- 外部 Football API を利用してデータを取得・表示

---

### 試合日程表示

- リーグ別の試合日程を表示
- 日付・リーグ単位での情報確認が可能
- Football API を利用したデータ取得

---

### チーム検索機能

- チーム名による検索機能
- 検索結果としてチームの基本情報を表示
- API のレスポンスデータを加工して画面に反映

---

### ガイドページ

- Web サイトの利用方法を説明するガイドページ
- 各機能の目的や使い方を整理
- 初めて利用するユーザーでも理解しやすい構成

---

## 使用技術および API 活用

### 外部 API

- サッカーニュース API
- Football API  
  - リーグ順位データ  
  - 試合日程データ  
  - チーム情報データ  

本プロジェクトでは、  
API を直接呼び出し、取得したレスポンスデータを加工し、  
画面に表示するまでの一連の処理を重点的に学習しました。

## まとめ

本プロジェクトは、  
サッカー情報を扱う掲示板型 Web アプリケーションを通じて、  
外部 API 連携、ログインベースのサービス構造、  
データ表示の流れを実践的に学習することを目的として開発しました。

単なる機能実装に留まらず、  
API を活用した Web サービス構築の基礎を  
体系的に理解することを重視しています。
